Technical Settings

File paths

In order to facilitate easy data entry you can enter custom paths to different files which you wish to enter into the database. CSV, XLSX, and JSON formats are accepted at this time. Simply copy paste the file path into the text box. For organization we recommend keeping the files in the same folder as the Scanner installation. Then click Update Database and the file will be added to the database. See database Schema for file formatting.

Caution

By clicking this button you will set the flag for PostgreSQL setup to false. Next time you launch the app you will be prompted to walk through the installation process. There are options to delete all Databases and start a clean install. Only reset the configuration settings. Or leave all databases. This option should only be used if there is a catastrophic error in the app.
